Asciimath using explicit block and macro names

Configuration

  • The :stem: attribute in the document header specifies latexmath.

  • In these examples, use of asciimath is signaled using the explicit ‘asciimath’ block and macro name notation.

Examples

Asciimath block

A matrix followed by some square roots can be written as

\$[[a,b],[c,d]]((n),(k)) sqrt(4) = 2 sqrt(9) = 3\$

Inline in a paragraph

Some text might suddenly need to show some matrix calculations, such as \$[[a,b],[c,d]]((n),(k))\$.

Inline, using attribute substitution

If you are not interested in exciting formulae such as \$[[a,b],[c,d]]((n),(k))\$, perhaps you would find the more mundane \$sqrt(4) = 2\$ or even \$sqrt(9) = 3\$ more to your taste.

Lists

  • A matrix can be written as \$[[a,b],[c,d]]((n),(k))\$.

    • A square root looks like \$sqrt(4) = 2\$

  • Another square root looks like \$sqrt(9) = 3\$

Section headings

A matrix can be written as \$[[a,b],[c,d]]((n),(k))\$.

A square root looks like \$sqrt(4) = 2\$.

Another square root looks like \$sqrt(9) = 3\$.
