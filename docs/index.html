Proyecto Delivery Ecommerce

Bienvenido a la documentación oficial del proyecto Delivery Ecommerce. Este sistema fue creado y documentado íntegramente por Jeremías Carrasco como solución ágil y profesional para la gestión de pedidos, usuarios y entregas en una tienda de comida al paso.

Característica Detalle

Autor

Jeremías Carrasco

Presupuesto

Bajo

Transacciones estimadas/día

10 (máximo 100)

Tecnología Backend

Go (API REST)

Tecnología Frontend

HTMX y JS Vanilla

Base de Datos

SQLite3 local

Caso de Uso Principal

Delivery Al Paso
Este proyecto busca resolver las necesidades de comercios pequeños, centralizando la gestión de pedidos en una interfaz intuitiva y rápida de implementar.
